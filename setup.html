üöÄ Quick Start

Choose your preferred method to run the application:

Option 1: Using the provided scripts

Windows: Double-click start-server.bat
Mac/Linux: Run ./start-server.sh

Option 2: Manual server start

Node.js: node server.js
Python: python -m http.server 8000
PHP: php -S localhost:8000

Option 3: VS Code Live Server

Install the Live Server extension and right-click on index.html

üîë API Configuration (Optional)

To enable full functionality with real music data:

  1. Copy config.example.js to config.js
  2. Sign up for free API keys:
    • Spotify API
    • Jamendo API
    • Genius API
  3. Add your API keys to config.js

Note: The app works without API keys using demo data!
